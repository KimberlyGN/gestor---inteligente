# Gestor Inteligente con Módulo IA

Aplicación web desarrollada con Flask que permite gestionar actividades y generar reportes en PDF con una interfaz moderna estilo "AI Box".

---

## Funcionalidades

- Agregar actividades
- Visualizar lista de actividades
- Generar reportes en PDF
- Gestión básica de productividad
- Interfaz moderna y minimalista

---

## Tecnologías utilizadas

- Python
- Flask
- ReportLab
- HTML
- CSS

---

## Instalación local

1. Clonar el repositorio:

git clone https://github.com/TU_USUARIO/TU_REPOSITORIO.git

2. Entrar al proyecto:

cd nombre-del-proyecto

3. Instalar dependencias:

pip install -r requirements.txt

4. Ejecutar la aplicación:

python app.py

5. Abrir en el navegador:

http://127.0.0.1:5000

---

## Deploy

Proyecto listo para desplegar en Render utilizando:

gunicorn app:app
